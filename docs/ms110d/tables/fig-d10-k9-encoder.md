## Prose (D.5.3.2.2)

A constraint length 9, rate 1/2 convolutional code shall be used prior to
puncturing. Figure D-10 is a pictorial representation of the encoder.

## Polynomials (as printed)

    (b0) T1 = x^8 + x^6 + x^5 + x^4 + 1
    (b1) T2 = x^8 + x^7 + x^6 + x^5 + x^3 + x^1 + 1

## Tap-by-tap wiring (read from the figure)

Shift register cells left to right: X^8, X^7, X^6, X^5, X^4, X^3, X^2, X, 1.
Input enters at the X^8 (leftmost) cell.

- Upper summing node → output b0 (polynomial T1): taps from cells
  **X^8, X^6, X^5, X^4, 1** (five taps; no taps from X^7, X^3, X^2, X).
- Lower summing node → output b1 (polynomial T2): taps from cells
  **X^8, X^7, X^6, X^5, X^3, X, 1** (seven taps; no taps from X^4 or X^2).

Both summing nodes are modulo-2 additions.

## Output order

"For each bit input to the encoder, two bits are taken from the encoder, with
the upper output bit b0, generated by polynomial T1(x), taken first."

## Octal form (derived, MSB = x^8 coefficient first)

- T1 (b0): 101110001 = 0o561
- T2 (b1): 111101011 = 0o753
